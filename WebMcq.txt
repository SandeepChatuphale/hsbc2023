
1) Which of the following keywords is used to define a variable in Javascript?

a) var
b) let
c) both a,b
d) None of the above

Answer:- c

2) Which of the following methods is used to access HTML elements using Javascript?
a) getElementById()
b) getElementsByClassName()
c) both a,b
d) None of the above

Answer : c

3) How can a datatype be declared to be a constant type?
a) var
b) let
c) const
d) constant

Answer:- c


4) What will be the output of the following code snippet?
<script type="text/javascript">
a = 5 + "9";
document.write(a);
</script>

a) Error
b) 14
c) 9
d) 58

Answer: - d


5) What will be the output of the following code snippet?
	<script type="text/javascript">
	let arr = [10,40,56];
	document.write(arr[-1]);
	</script>

a) Error
b) undefined
c) null
d) None of the above

Answer: - b
